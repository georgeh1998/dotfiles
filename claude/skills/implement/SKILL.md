# Implement スキル

`/implement` コマンドに応答して、`claude/tasks/plan.md` の実装計画を順番に実行するスキル。

## ワークフロー

### 1. plan.md を読む

`claude/tasks/plan.md` を読み、実装タスクの一覧を把握する。存在しない場合は `/plan` を先に実行するよう伝えて終了。

### 2. タスクを順番に実行する

全タスクを最後まで一気に実行する。各タスクについて：

1. タスク名と概要を表示する
2. Edit/Write ツールで変更を実行する
3. 変更したファイルを簡潔に報告して次へ進む

問題が発生した場合のみ止まってユーザーに相談する。

### 3. 完了時

全タスク完了後、変更したファイルの一覧をまとめてユーザーに報告する。

## 注意事項

- 計画に書かれていない変更は加えない
